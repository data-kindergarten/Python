### 개념

- 벡터와 행렬을 포함하는 개념
- 💡 벡터(Vector) : 1차원 배열(array)
- [💡](https://ko.wikipedia.org/wiki/%ED%96%89%EB%A0%AC) 행렬(Matrix) : 2차원 배열(array)

### **1차원 배열 (벡터)**

```python
# 숫자, '문자', 숫자+'문자' 모두 입력가능

# 숫자
A = np.array([1, 2, 3, 4, 5])
A
array([1, 2, 3, 4, 5])

# 문자

# 숫자+'문자' : 숫자가 문자로 인식됨

```

### 2차원 배열 (행렬)

```python
A2 = np.arrary([[1, 2, 3], [4, 5, 6]]) # 2 x 3 array
A2
array([[1, 2, 3],
      [4, 5, 6]])
```

### 3차원 배열

```python
A3 = np.array([[[1, 2, 3],[4, 5, 6], [[10, 20, 30].[40,50,60]]])
A3
array([[[ 1,  2,  3],
        [ 4,  5,  6]],

       [[10, 20, 30],
        [40, 50, 60]]])               
```
